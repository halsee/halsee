Objective

Redesign and develop alcatrazhistory.com to be mobile-friendly and easier to navigate.

Result

A redesigned, restructured, mobile-friendly website resulting in X% more page views and X% longer sessions.

Description

In curating content for the Alcatraz Island guide of the Guidekick app, we became good friends with Michael Esslinger, the foremost historian of Alcatraz. This was an exceptionally interesting project to work on because of the stories surrounding Alcatraz. I won’t give any spoilers, but Alcatraz has been much more than just a prison for hardened criminals. If you’re interested in the full history, look no further, you’ve found it! The sheer amount of content might be a bit overwhelming, but the website’s intent is to be a historical archive, and it succeeds wildly in that.
