Ambiarc Map Platform

Ambiarc Map Platform

Objective

Create a tool which makes modeling, managing, and integrating an indoor 3D map easy.

Result

Ambiarc! It literally does all of those things listed above.

Description

Ambiarc was created partly out of a necessity to make the creation of our own Guidekick apps more efficient. It was also created with the intent of becoming a platform that others could use to create and manage 3D maps. Previously, we were custom modeling, texturing, and managing each 3D map for each guide and app. While the maps looked fabulous, we needed a more streamlined process and quicker way to get a map created and functional within each app. As it exists right now, you feed Ambiarc a single, or set of, GeoJSON files derived from floor plans. With that data it automatically maps out the space in 3D. Import that map into a custom tool we created, and you can do all sorts of neat (and useful) things like add points to the map, create labels, add interactivity, change the color scheme, and a lot more. The map is platform agnostic, so it can be integrated into a project for web, mobile, touchscreen, or even augmented or virtual reality. A Javascript SDK allows the map experience to be customized even further. Full documentation // Github
